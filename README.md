# 📊 Supply Chain Efficiency & Inventory Optimization Dashboard

## 🚀 Overview
This project is a Power BI dashboard designed to analyze and optimize supply chain performance and inventory management. It provides insights into stock levels, supplier efficiency, and demand forecasting.

---

## 🎯 Objective
- Monitor inventory performance
- Reduce stockouts and overstocking
- Improve supplier delivery performance
- Enable data-driven decision-making

---

## 🛠 Tools Used
- Power BI
- Excel
- DAX (Data Analysis Expressions)
- Power Query

---

## 📂 Dataset Features
- Product & Category details
- Supplier & Warehouse data
- Inventory levels (Opening, Closing, Incoming)
- Sales (Units Sold)
- Delivery Status & Lead Time
- Demand Forecast vs Actual

---

## ⚙️ Key KPIs
- Total Inventory
- Stockout Count
- On-Time Delivery %
- Forecast Accuracy %
- Inventory Turnover Ratio

---

## 📊 Dashboard Features
- KPI Cards for quick insights
- Supplier performance analysis
- Inventory trends visualization
- Demand vs Forecast comparison
- Interactive filters (Region, Category, Supplier)

---

## 🔍 Key Insights
- Identifies stockout-prone products
- Highlights delayed suppliers
- Tracks demand forecasting accuracy
- Helps optimize inventory levels
